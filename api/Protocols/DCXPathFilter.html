DCXPathFilter protocol reference

DCXPathFilter.h

Overview

A filter is called once for each node that will be added to the new current list of nodes.

It provides the ability to do additional processing based on the node or it's index within it's parent node.

Instance methods

acceptNode:index:errorVar:

This method is called for each node added.

- (BOOL)acceptNode:(DCDMNode *) node index:(int) anIndex errorVar:(NSError **) aErrorVar
Parameters
node

A reference to the node being added. Can be any type of node.

anIndex

The index of the node within the list of returned nodes from a selector. This is 0 based.

aErrorVar

A reference to a variable where an error can be placed if it occurs.

Return value

YES if the node should be added to the results, NO if it should not.
