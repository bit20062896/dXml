NSObject(SoapTemplates) category reference

NSObject+SoapTemplates.h

Overview

Adds the ability to generate a Soap message template to any class.

This is useful for classes wishing to generate soap web service messages.

Tasks

Templates

  • - createBasicSoapDM Returns a DCXmlDocument which contains the basic soap message elements.
  • - createSoapFaultDM Returns a DCXmlDocument which contains a standard soap fault structure.

createBasicSoapDM

Returns a DCXmlDocument which contains the basic soap message elements.

- (DCXmlDocument *)createBasicSoapDM
Discussion

Essentially this creates the Evelope, Header and Body elements. Here is a copy of the output from this method

 <?xml version="1.0" encoding="UTF-8"?> <soapenv:Envelope xmlns:xsd="http://www.w3.org/2001/XMLSchema"   xmlns:xsi="http://www.w3.org/2001/XMLSchema-instance"   xmlns:soapenv="http://schemas.xmlsoap.org/soap/envelope/">   <soapenv:Header />   <soapenv:Body /> </soapenv:Envelope>
