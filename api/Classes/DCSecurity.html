DCSecurity class reference

NSObject
DCSecurity.h

Overview

Security is the main factory class for applying security.

Essentially you set it up with the security information you have then ask it to provide a security handler back. This handler can then be used to implement that style of security on the soap messages. Here is an example of how to use it:

 DCSecurity *security = [DCSecurity createSecurityWithUserid: userid password: password]; NSObject <SecurityModel&ht; *securer = [security createSecurityModelOfType: securityType]; [securer secureSoapMessage: soapMsg];
Security seperated into two parts to make it easy to use. The act of constructing an instance if Security also gives it the necessary data it needs to work with. ie.userid and password. The second part of getting the model and applying it to the message provides the flexibility of being able to aply different models as needed.
