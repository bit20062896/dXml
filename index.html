dXml by drekka

iPhone static library providing xml parsing into document models, XPath expressions, xml messaging to/from servers SOAP messaging to/from web service providers, SOAP fault handling.

Dependencies

If you just want to use the static library then download a compiled dmg file from here.

If you want to compile this library then you can download a source tar or zip or checkout from github.
