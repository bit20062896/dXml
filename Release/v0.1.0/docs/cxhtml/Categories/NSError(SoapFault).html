NSError(SoapFault) category reference

NSError+SoapFault.h

Overview

This category adds functionality to the NSError class for dealing with soap faults that have been returned as NSError instances.

Tasks

Soap message handling

  • - isSoapFault Returns YES if the NSError contains soap fault information.
  • - soapFaultMessage returns the contents of the faultMessage userinfo value.
  • - soapFaultCode returns the contents of the faultCode userinfo value.

Soap fault factory methods

  • + errorWithSoapResponse: Factory method which creates a NSError containing a soap fault.
  • + errorWithSoapFault: Factory method which creates a NSError containing a soap fault.
