This category adds xpath capability to the DCXmlNode.

  • - xmlNodeFromXPath: Returns the DCXmlNode specified by the supplied Xpath.

Instance methods

xmlNodeFromXPath:

Returns the DCXmlNode specified by the supplied Xpath.

- (DCXmlNode *)xmlNodeFromXPath:(NSString *) aXpath
Discussion

If there is no node at any point in the xpath a nil is returned.

valueFromXPath:

Returns NSString specified by the supplied Xpath.

- (NSString *)valueFromXPath:(NSString *) aXpath
Discussion

If there is no node at any point in the xpath, or the end of the xpath is not a DCTextNode, then a nil is returned.
