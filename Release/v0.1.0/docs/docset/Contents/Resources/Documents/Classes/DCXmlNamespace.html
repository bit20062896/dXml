DCXmlNamespace class reference

NSObject
DCXmlNamespace.h

Overview

Represents the prefix and url of a namespace associated with an DCXmlNode.

For example in 

 <abc xmlns:def="http://ghi" />
"def" is the prefix, and "http://ghi" is the url of the namespace. Generally speak urls for namespaces always take the form of a http:// url.
