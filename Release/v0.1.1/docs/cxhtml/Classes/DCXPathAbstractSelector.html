DCXPathAbstractSelector class reference

NSObject
DCXPathAbstractSelector.h

Overview

Abstract parent class of all selectors.

The basic idea of a selector is that it is presented with a node and asked to select further nodes based on that node. For example a parent node select is given a node and returns it's parent. In addition to this, selectors can also have multiple filters added to them. If a selector has filters, before adding a node to the list of ndoes to be returned, each filter is asked if it is ok to add that node. Only if all filters agree, will the node be added.

Tasks

Other

  • - addFilter: Adds the passed filter to the list of filters.
  • - selectFromNode:index:errorVar: Called for each node in the current list of nodes by DCXPathExecutor.
  • - selectNodesFromNode:index: Called internally to obtain the nodes for the selector.

selectFromNode:index:errorVar:

Called for each node in the current list of nodes by DCXPathExecutor.

- (NSArray *)selectFromNode:(DCXmlNode *) currentNode index:(int) anIndex errorVar:(NSError **) aErrorVar
Parameters
currentNode

The current DCXmlNode to be used as the base node for the current xpath token.

anIndex

The index of the parent node in the list of current nodes being processed.

aErrorVar

A reference to a variable that will be populated with a NSError should something go wrong with trying to apply the xpath to the document model.

Return value

An NSArray containing the nodes found by the selector. Unless an error is found, in which case this is nil.

Discussion

Empty implementation.

This should not be overridden because it contains the common processing applicable to all selectors.

selectNodesFromNode:index:

Called internally to obtain the nodes for the selector.

- (NSArray *)selectNodesFromNode:(DCXmlNode *) node index:(int) anIndex
Parameters
node

The node current being processed. It is the based from which the selector shoul select new nodes.

anIndex

The index of the current node in the list of nodes being processed.

Return value

A NSArray of the nodes being selected. Note, as per the filter details, these nodes may not make it back to the DCXPathExecutor.

Discussion

Empty implementation designed to be overridden.

Derived classes should implement this method to return the selected nodes.
