XmlSubtreeParserDelegate class reference

NSObject
XmlSubtreeParserDelegate.h

Overview

Main class for parsing xml streams.

This class is capable of receiving and processing the incoming xml events and assembling from them a complete DM data graph. IT responds to the standard NSXMLParser delegate methods to achieve this.
