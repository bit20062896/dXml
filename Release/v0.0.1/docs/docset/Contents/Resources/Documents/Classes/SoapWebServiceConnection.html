SoapWebServiceConnection class reference

UrlConnection : NSObject
SoapWebServiceConnection.h

Overview

Extension of the UrlConnection class which is specially designed to handle soap web service calls to servers.

This class is the main class of the dXml API. Here's an example of using it 

 // Simple request for data with no security. NSString *xml = @"<dhc:balance xmlns:dhc=\"" MODEL_SCHEMA "\">"                 @"<forAccountNumber>1234</forAccountNumber>"                 @"</dhc:balance>";   // Create the service. NSString *url = @"http://localhost:8080/services/Banking"; NSString *action = @"\"http://www.dhcbank.com/banking/balance\"";SoapWebServiceConnection *service = [SoapWebServiceConnectioncreateWithUrl: url soapAction: action];   // And call it.WebServiceResponse *response = [service postXmlStringPayload: xml];   // And do something with the response.
