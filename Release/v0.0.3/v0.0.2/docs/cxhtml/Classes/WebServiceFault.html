WebServiceFault class reference

WebServiceResponse : NSObject
WebServiceFault.h

Overview

Returned from a call to a web service when a soap fault occurs at the server end.

Tasks

Other

  • + createWithFaultCode:: Factory for internal use that creates a web service fault with a passed code and message.

Other

  • - faultCode Gives direct access to the value of the soap fault code.
  • - message Gives direct access to the value of the soap fault message.
  • - javaException If present, will return the XmlNode containing a server generated Java exception.

Class methods

createWithFaultCode::

Factory for internal use that creates a web service fault with a passed code and message.

+ (WebServiceFault *)createWithFaultCode:(NSString *) message :(NSString *) aMessage
Discussion

generally this is used by internal classes which have caught an exception. We do this as returning faults is more efficient that throwing exceptions.
